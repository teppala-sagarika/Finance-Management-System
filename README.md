# 💰 Finance Management System

This is a full-stack web application developed to manage personal finances by tracking income and expenses. The application allows users to register, log in, add financial records, and view a dashboard with summaries and charts for better financial insights.

---

## 🚀 Features

* User Registration and Login
* Add Income and Expense Records
* Dashboard with total income, expenses, and balance
* Data visualization using charts (Pie, Bar, Line)
* Filter records by type and category

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite), Axios, Chart.js
* **Backend:** Node.js, Express.js
* **Architecture:** MVC pattern (Models, Controllers, Services, Routes)

---

## ⚙️ Setup Instructions

### 1. Clone the repository

```
git clone https://github.com/your-username/Finance-Management-System.git
```

### 2. Install dependencies

**Frontend:**

```
cd frontend
npm install
npm run dev
```

**Backend:**

```
cd backend
npm install
npm start
```
